# CAPI (cocktail-API)

Fetch cocktail recipes with this API.


/api/cocktails/         GET
/api/cocktails/:id      GET
/api/cocktails/         POST
/api/cocktails/:id      PUT
/api/cocktails/:id      DELETE

/api/zutaten/           GET         +

/api/zutaten/           POST        +
/api/zutaten/:id        DELETE      +
